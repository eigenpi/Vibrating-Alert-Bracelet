Top3.prt and bottom3.prt are the same as the stl files named Senior_Design_Top_Final_Rev
 and Senior_Design_Bottom_Final_Rev. The assembly part contains both the top, bottom, board 
and battery in an assembly. The board2.0 part is the 3d mock up of the board with major
components and to the exact measured size of the pcb.
Program used to create parts was NX 8.5
